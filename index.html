Git Commands Mastered

1. Repository Creation & Cloning

# Creating new repository git init flexisaf-version-control cd flexisaf-version-control/span> echo "# Git Learning Project" > README.md git add README.md git commit -m "Initial commit" # Cloning repository git clone git@github.com:frontendhashira/flexisaf-version-control.git

2. Branch Operations

# Creating and switching branches git branch feature/html-structure git switch feature/html-structure git switch -c feature/css-styling # Managing branches git branch -a # List all branches git branch -m old-name new-name # Rename git branch -d feature-name # Delete branch

3. Committing & Reverting

# Staging and committing git add index.html git add . git commit -m "feat: add homepage structure" git commit -am "fix: update existing files" # Reverting commits git revert HEAD git reset --soft HEAD~1 git reset --hard HEAD~1

4. Remote Operations (Upstream/Downstream)

# Setting up remote git remote add origin https://github.com/frontendhashira/flexisaf-version-control.git git remote -v # View remotes # Pushing upstream git push origin main git push -u origin feature/new-feature git push --all # Push all branches # Pulling downstream git pull origin main git fetch origin git fetch --all

5. Merging & Pull Requests

# Merging branches git checkout main git merge feature/html-structure git merge --no-ff feature/css-styling # Handling conflicts git status # Check conflict files # Edit files to resolve conflicts git add resolved-file.txt git commit -m "resolve: merge conflicts"
